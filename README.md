# Java Full-Stack Notes

Welcome to the Java Full-Stack Notes repository! This repository is dedicated to storing study notes and tutorials for learning Java full-stack development, starting from Core Java basics to advanced topics including Spring Boot, frontend technologies, and databases.

## Repository Structure

- **Core_Java**: Fundamental Java programming concepts and features.
- **Advanced_Java**: Topics like JDBC, Servlets, JSP, and design patterns.
- **Spring_Boot**: Concepts and tutorials on Spring Boot framework.
- **Frontend**: Basics of HTML, CSS, JavaScript, React, and integration with backend.
- **Database**: Notes on MySQL, JPA, Hibernate, and query optimization.

## How to Use

- Navigate the folders to find notes on specific topics.
- Each topic is documented in Markdown files for easy reading and editing.
- Feel free to contribute by creating pull requests or opening issues.
